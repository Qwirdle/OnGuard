Public Wi-Fi, although easily accessible, carries many risks. Cyber criminals can easily commit acts of data theft, malware, and gain unauthorized access to you or any organizations you are a part of through public wifi. Although it is very commonly done and easy to connect to public Wi-Fi, doing so is something that one should quit to avoid unnecessary consequences, at least without using proper tools.

Many cyber attacks are done over public Wi-Fi networks. The ease of access to other devices allows cyber criminals to decrypt user traffic like texts, among other attacks. Cyber criminals can even set up fake public networks in order to trick people into connecting into networks where the criminals can then easily access the victims data from. One common attack committed over public Wi-Fi networks is a man in the middle attack, where a third party intercepts a message being sent between two computers, usually for malicious reasons.

Despite all this, there are some countermeasures one can take when using public Wi-Fi to mitigate these risks:

  • Use Virtual Private Network's (VPN's): VPNs encrypt one’s internet traffic, making it harder for anyone other than you to access. Many free and paid ones exist, though double check with your IT department before installing one on a work device.

  • Don’t interact with sensitive data: Avoid logging into important accounts, or responding to important emails on public wifi network as you never know if someones watching.

  • Turn off file sharing: File sharing on public wifi networks could allow for cyber criminals to have easy access to your files.

  • Don’t setup automatic connection to public networks: You never want to be unknowingly connected to a public WiFi networkbecause of the potential dangers, which can be hindered by activating automatic connection to a public network because then connection will happen without you knowing.

  • Verify Network Authenticity: Ask a venews clerk if a public network is the one that belongs to that store, as sometimes cyber criminals set up fake public networks to mimic that of venus in order to attract potential victims.

  • Maintain basic security: Keeping your system updated, login secured, and following the other important steps towards maintaining basic security that are described in this course create an important buffer against cybercrime in most if not all situations.

Summary

  • Public Wi-Fi poses risks like data theft, malware, and unauthorized access.

  • Use a VPN to encrypt traffic.

  • Avoid sensitive transactions and turn off file sharing.

  • Disable automatic connections and verify network authenticity.

  • Keep devices updated and maintain strong security practices.
